# Job Application Tracker

A Kanban board for tracking job applications, from "eyes on" through offer. Free, local-first (works without an account), built with React + Vite + TypeScript + Tailwind, dnd-kit, and Supabase.

Full spec: [`job-tracker-mvp-brief.md`](./job-tracker-mvp-brief.md). Build progress and milestone status: [`PLAN.md`](./PLAN.md).

## Development

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` and fill in your Supabase project URL and anon key (see Project Settings → API in the Supabase dashboard).
